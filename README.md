# did:ethr Manager

Static app for resolving and managing `did:ethr` identifiers.

Supports:
- address-based `did:ethr` DIDs
- compressed secp256k1 public-key `did:ethr` DIDs
- DID document resolution
- key, service, owner, and deactivate operations via ERC1056

[Live demo](https://mirceanis.xyz/ethr-manager/)

Local use:
- serve the repo as a static site
- open it in a browser with an EIP-1193 wallet installed
